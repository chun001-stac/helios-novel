# helios-novel

一个中度科幻长篇小说项目。

## 核心方向

故事以“赫利俄斯事件”为起点，围绕人类科技发展、宇宙探索、古老文明遗迹与异常信号展开。

前期不直接接触外星文明，而是通过遗迹、信号、历史断层和技术异常逐步揭开宇宙真相。中后期再慢慢引入外星文明，并把“伊洛文明最后发现了什么，以及他们为什么选择离开宇宙”作为长期悬疑推进。

## 初步结构

- 第一部：赫利俄斯事件与人类联邦时代的开端
- 第二部：伊洛文明遗迹、异常信号与宇宙深层秘密
- 第三部：人类文明的选择，以及宇宙边界之外的答案

## 目录说明

- `chapters/`：正文草稿
- `characters/`：核心群像人物设定
- `worldbuilding/`：世界观、科技、文明、遗迹、时间线
- `docs/`：大纲、章节规划、设定总表
- `notes/`：灵感、临时片段、待整理材料

## 写作原则

- 没有单一主角，采用核心群像推进。
- 科技发展与宇宙探索并重。
- 神秘哲学感逐步加深，但不削弱硬科幻逻辑。
- 让公司、联邦政府、科研机构和文明遗迹共同构成主要叙事力量。
