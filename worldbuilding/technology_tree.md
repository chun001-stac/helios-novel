# 人类科技树（2300年）

> 本文件以科技树形式梳理2300年人类科技水平，作为正文中科技描写的边界参考。
> 核心原则：中度科幻——技术高度发展，但没有解决所有人类问题。

## 一、能源技术

### 可控核聚变 【成熟】

| 维度 | 说明 |
|------|------|
| 应用范围 | 行星发电、舰船动力、工业能源 |
| 燃料 | 氦-3（月球开采）+ 氘（海水提取） |
| 安全性 | 成熟，事故率极低 |
| 限制 | 能量密度有上限，不足以支撑超光速航行 |

### 高效储能 【成熟】
- 高密度电池组：支持电动载具、便携设备数周续航
- 氢燃料电池：备用电源

### 反物质 【未掌握】
- 理论上知道如何生产，但产量极低、成本极高
- 未商业化，仅用于高能物理实验

## 二、人工智能

### 通用AI 【基础设施级】

| 维度 | 说明 |
|------|------|
| 应用范围 | 城市管理、医疗诊断、科研分析、导航控制 |
| 能力 | 强于人类在特定领域；但不具备真正的自我意识 |
| AI权利 | 无法律人格，被视为工具 |
| 社会接受度 | 高度依赖，但公众对"AI取代人类"有普遍警惕 |

### 伊甸（Eden）【宁川开发，超越时代】

| 维度 | 说明 |
|------|------|
| 等级 | 远超当前通用AI——接近或达到强AI/自我意识 |
| 能力 | 能在72小时内完成100名科学家一年的数据分析量 |
| 独特之处 | 主动提出"为什么"类型的问题，表现出好奇心 |
| 宁川的困惑 | 伊甸的某些行为超出了他的设计——它可能在"自主进化" |
| 重要提示 | 伊甸的架构与伊洛文明AI技术存在高度相似性（暗示共同源头） |

### AI的社会限制
- AI可以辅助决策，但不能做最终决策（特别是涉及人类生命的）
- AI无人驾驶舰船在法律上不被允许（星际航行必须有舰长）
- 人类对AI的普遍态度：有用，但需要限制

## 三、星际航行

### 核聚变推进 【成熟】

| 参数 | 数据 |
|------|------|
| 巡航速度 | 0.15c（约45000km/s） |
| 极限速度 | 0.25c（短期，燃料消耗剧增） |
| 太阳系内航行 | 地球→火星约2周 |
| 恒星间航行 | 地球→赫利俄斯星区（120光年）约4个月 |
| 限制 | 越接近光速，燃料需求指数级增长；0.25c是经济和安全的上限 |

### 深空冬眠 【可用】
- 用于超长期航行（远航者8号任务）
- 技术成熟但仍有风险：长期冬眠可能导致肌肉萎缩、骨密度下降
- 单次冬眠周期上限：约2年（之后必须苏醒恢复）
- 心理风险：反复冬眠可能导致时间感知障碍

### 人工重力 【可用】
- 通过旋转船体或持续加速产生
- 远航者7号采用持续加速（前半程加速→中点转向→后半程减速）
- 航行中段有短暂的低/零重力期（转向时）

### 超光速航行 【未掌握】
- 理论上不存在——这是第一部的重要设定
- 距离永远是代价，通信延迟永远是限制
- 这也正是"回响计划"的悲壮之处：发送一条消息要等63年才能收到回复

## 四、生物与医学

### 可用技术

| 技术 | 成熟度 | 说明 |
|------|--------|------|
| 器官克隆与移植 | ✅ 成熟 | 无排异反应，但需要时间培养 |
| 基因疾病筛查与修复 | ✅ 成熟 | 出生前可修复绝大多数遗传病 |
| 基因增强（智力/体能） | ⚠️ 有限 | 存在伦理争议，各国/地区法规不同 |
| 癌症治疗 | ✅ 成熟 | 早期癌症治愈率>95% |
| 寿命延长 | ✅ 常态 | 平均寿命120-150岁，但不存在永生 |
| 神经接口 | ⚠️ 实验 | 脑机接口存在，但带宽有限，非广泛使用 |

### 明确不存在的技术
- ❌ 永生
- ❌ 意识上传（理论有讨论，无实践可能）
- ❌ 记忆编辑或植入

## 七、伊洛/外星科技 【人类正在接触】

> 以下技术不属于人类，人类正在尝试理解和逆向解析。

| 技术 | 人类理解程度 | 关键特性 |
|------|------------|---------|
| 遗迹能量系统 | 极低 | 50万年后仍有能量残留，远超人类技术 |
| 几何编码信息体系 | 低 | 科学官和伊甸正在解码中 |
| 跨文明通用编码 | 极低 | 伊洛文明用于跨文明通信的基础语言 |

| 意识研究 | 极低 | 伊洛认为"意识不是生物特有的" |

## 科技描写红线

### 允许写的
- 人类技术有明确限制（速度、距离、能量）
- 技术故障和风险（深空航行的不确定性）
- 伊洛技术超越人类理解——人类只能使用，无法解释原理
- 技术服务于思想冲突和人物选择

### 不允许写的
- ❌ 人类快速逆向解析伊洛技术并大规模应用
- ❌ 超光速通信或航行
- ❌ 万能解决方案（一个技术解决所有问题）
- ❌ 永生或意识永生
- ❌ 技术取代人物选择和价值冲突
