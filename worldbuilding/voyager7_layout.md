# 远航者7号（Voyager-7）内部布局

> 为正文中舰船场景的空间一致性提供参考。

---

## 总体参数

| 项目 | 数据 |
|------|------|
| 全长 | 约280米 |
| 最大直径 | 约60米 |
| 结构 | 三段式：前段指挥区、中段生活区、后段动力与工程区 |
| 船员 | 7名核心 + 2名安保 |
| 自持力 | 最长18个月（含循环系统） |
| 最高速度 | 0.15c（巡航）/ 0.25c（极限） |
| 船体材料 | 碳化硅复合材料 + 多层防护壳 |

---

## 前段：指挥区

### 舰桥（Bridge）
- 位置：船体最前端
- 面积：约40m²，半圆形布局
- 配置：
  - 中央指挥席（林默位置）
  - 左侧航行控制台（阳子位置）
  - 右侧系统监控台（张伟明位置）
  - 前方全景显示幕（可切换星图/传感器/通信界面）
- 特点：深空航行时照明偏暗，以屏幕光为主

### 通信室（Communications Room）
- 位置：舰桥后方右侧
- 面积：约15m²
- 配置：
  - 深空通信阵列控制台（奥布莱恩位置）
  - 数据加密/解密终端
  - 信号分析工作站

### 导航室（Navigation Room）
- 位置：舰桥后方左侧
- 面积：约12m²
- 配置：
  - 星图工作站
  - 航线规划终端
  - 引力异常监测仪（——正是这里最早注意到HZ-431的异常）

### 科学实验室（Science Lab）
- 位置：前段中层，紧邻舰桥下方
- 面积：约30m²
- 配置：
  - 样本分析台
  - 全息投影台
  - 光谱/质谱分析仪
  - 小型材料测试设备
  - 数据存储服务器
- 特点：帕特尔大部分时间在此处工作；遗迹数据的第一处理地点

### 观测舱（Observation Dome）
- 位置：前段最前端，舰桥上方凸起半球
- 面积：约8m²
- 配置：
  - 360度透明穹顶（可切换遮光模式）
  - 两把观测椅
  - 手动光学望远镜接口
- 用途：深空目视观测、心理放松、重要时刻的独处空间

## 中段：生活区

### 船员舱（Crew Quarters）
- 位置：中段环形分布，共6间
- 每间面积：约10m²
- 配置：
  - 单人床（可折叠收纳）
  - 书桌/工作台
  - 个人储物柜
  - 小型全息屏幕
  - 独立的微型卫生间
- 分配：
  - 01号：林默（靠近前段一侧）
  - 02号：阳子
  - 03号：张伟明
  - 04号：帕特尔（靠近实验室一侧）
  - 05号：沃尔科娃（靠近医疗室一侧）
  - 06号：奥布莱恩（靠近通信室一侧）
  - 门德斯及安保人员住在后段改造舱室

### 医疗室（Medical Bay）
- 位置：中段中部偏前
- 面积：约20m²
- 配置：
  - 两张医疗床
  - 自动诊断台
  - 手术辅助机械臂
  - 药品/器材柜
  - 心理健康咨询区（一张椅子 + 一张躺椅）

### 食堂/休息区（Mess & Common Area）
- 位置：中段中心区域
- 面积：约35m²
- 配置：
  - 长桌 x1（可坐8人）
  - 食品加热/制备台
  - 饮料机
  - 小型书架（实体书 + 电子阅读器）
  - 墙上可切换显示地球/火星风景
- 特点：日常会议和非正式讨论的主要场所

### 水培舱（Hydroponics Bay）
- 位置：中段后端
- 面积：约15m²
- 用途：种植新鲜蔬菜/香料，辅助空气循环
- 特点：少量绿色，是船上最"有生命气息"的地方

---

## 后段：工程区

### 工程中心（Engineering Core）
- 位置：后段前端
- 面积：约25m²
- 配置：
  - 主控台（张伟明主要工作位置）
  - 动力系统监控
  - 生命维持系统面板
  - 全船系统诊断终端
- 特点：温度比前段高2-3°C，有持续的机械低频嗡鸣

### 探索艇舱（Scout Pod Bay）
- 位置：前段与中段之间的腹部，独立对接结构
- 容量：2艘探索艇
- 探索艇参数：
  - 小型：载3人，航程5000km
  - 用途：遗迹内部探测、行星表面着陆

### 气闸（Airlock）
- 位置：前段腹部（探索艇舱入口） + 后段尾部（外部检修口）
- 用途：EVA出舱、外部设备部署

## 关键场景指引

| 场景 | 发生位置 | 备注 |
|------|---------|------|
| 航行日常 | 舰桥 / 食堂 / 船员舱 | 根据角色位置选择 |
| 团队会议 | 食堂 | 非正式；舰桥的中央指挥席周围 | 正式 |
| 林默独处 | 观测舱 / 个人舱 | 沉思场景 |
| 发现异常 | 导航室 / 科学实验室 | 第一阶段 |
| 遗迹数据解码 | 科学实验室 | 通讯室辅助 |
| 进入遗迹准备 | 探索艇舱 | 装备检查 |
| 私密通信 | 通信室 / 个人舱 | 与地球的延迟通话 |

## 与远航者8号的差异提示

远航者8号（第三卷启航）比7号更大，增加了：
- 更长的自持力（设计目标5年）
- 更大的科学实验室
- 改进的冬眠/深度休眠装置（具体技术路线待定）
- 集成伊甸子系统（伊甸-β）
