## 术语使用规则

1. **正文中优先使用中文名**，如"林默"而非"Lin Mo"
2. **专有名词首次出现时**，可括号标注英文，如"远航集团（Voyager Corporation）"
3. **对话中的人物称呼**：根据人物关系使用——顾远山可称"顾先生"或"Victor"，艾琳直接用"艾琳"
4. **舰船名**：正文中使用"远航者7号"，不加引号
5. **遗迹名**：正文中首次出现时用全称"黎明之环（Dawn Ring）"，之后可用简称"黎明之环"
