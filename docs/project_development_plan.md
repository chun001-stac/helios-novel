# Helios Novel Project Development Plan

## 项目定位

《赫利俄斯之门》是一个中度科幻、群像、哲学探索型长篇小说项目。

核心不是单一英雄，而是人类文明面对宇宙未知时的选择。

---

# 第一阶段：基础设定（当前阶段）

目标：建立小说世界数据库。

内容：

- 小说核心原则
- 人类文明设定
- 地球联邦结构
- 企业体系
- 科技水平
- 星际探索规则
- 伊洛文明档案
- 七个遗迹设定
- 核心人物档案

---

# 第二阶段：第一部剧情设计

作品：

《赫利俄斯之门》

目标：

描述人类第一次发现失落文明，并开启宇宙探索时代。

需要建立：

- 五卷结构
- 主要剧情节点
- 悬疑线索表
- 人物成长路线

---

# 第三阶段：三部曲详细规划

第一部：
赫利俄斯之门

第二部：
沉默文明

第三部：
第一文明

---

# 第四阶段：正文生成

规则：

- 按章节生成
- 每章经过一致性检查
- 所有设定必须来源于世界数据库
- 重大剧情变更需要更新设定文件

---

# 写作原则

1. 没有传统意义上的唯一主角。
2. 文明本身是叙事主体。
3. 科技用于探索思想问题。
4. 外星文明不是简单敌人。
5. 遗迹提供问题，而不是简单答案。
