# 第一部《赫利俄斯之门》写作风格指南

> 本文件作为正文写作的风格约束，确保整部作品语气、节奏和叙事方式保持一致。

---

## 一、叙述视角

### 核心规则

采用 **第三人称有限视角**，每章锁定一个角色的视角。

### 具体规范

- **每章有且仅有一个视角人物**
- 读者只能知道该视角人物知道的信息
- 不随意切换到其他角色的内心活动
- 章与章之间可以切换视角人物

### 视角分配建议

| 卷 | 主要视角 | 辅助视角 |
|----|---------|---------|
| 第一卷 | 林默、艾琳 | 顾远山、苏妍、宁川 |
| 第二卷 | 林默、林清寒、苏妍 | 顾远山、宁川、艾琳 |
| 第三卷 | 林清寒、艾琳、顾远山 | 宁川、沈岚 |
| 第四卷 | 苏妍、宁川 | 艾琳、沈岚、林默（远程） |
| 第五卷 | 宁川、林清寒、艾琳 | 林默（远程）、苏妍、顾远山 |

### 避免

- ❌ 上帝视角（直接告诉读者"他不知道的是……"）
- ❌ 在同一章内自由切换多个角色的内心
- ✅ 如果需要在同一场景中传递其他角色的信息，通过对话、动作和观察来暗示

## 二、章节结构

### 标准章节模板

```
【开篇】（1-2段）
  场景设定 + 视角人物当前状态

【发展】（主体）
  冲突/发现/对话推进

【转折】（靠近结尾处）
  新信息出现 / 决定做出 / 意外发生

【结尾钩子】（最后1-2段）
  制造继续阅读的欲望——问题、悬念、决定
```

### 章节长度

| 类型 | 建议字数 | 频率 |
|------|---------|------|
| 普通推进章 | 2500-3500字 | 多数章节 |
| 关键转折章 | 3500-5000字 | 每卷5-8章 |
| 高潮章 | 5000-6000字 | 每卷末1-2章 |
| 过渡章 | 1500-2500字 | 少量，用于调整节奏 |

### 每卷章节节奏

```
推进期（60%章节）→ 积累期（25%）→ 高潮期（15%）
```
- 推进期：日常、对话、准备
- 积累期：矛盾升级、新发现、压力增加
- 高潮期：关键事件、重大揭示、决策

## 三、语言风格

### 基调

- **克制、冷静** — 不过度煽情，让情节和对话本身传递情感
- **精确、具象** — 描写科技和宇宙场景时用具体细节代替抽象形容
- **留白、暗示** — 重要的哲学思考留给读者自己完成，不要用角色长篇大论来解释

### 具体规范

| 场景类型 | 语言风格 | 示例方向 |
|---------|---------|---------|
| 日常对话 | 自然、生活化 | 像普通人说话，不用过于文艺 |
| 科技描述 | 精确、简洁 | "直径约1000公里的环形结构"而非"巨大的、令人窒息的环形结构" |
| 宇宙场景 | 冷静、宏大 | 用对比和尺度制造震撼（人类vs遗迹的尺寸对比） |
| 情感场景 | 克制、留白 | 少写"他很悲伤"，多写动作和环境暗示 |
| 哲学思考 | 简洁、开放 | 不提问题答案，只提问题本身 |

### 禁止

- ❌ 过度华丽的形容词堆砌
- ❌ 角色长篇独白解释主题
- ❌ 插入作者评论或说教
- ❌ 网络用语、过度口语化的现代梗

---

## 四、对话写作规范

### 对话比例

- 每章对话占比：建议 **40%-60%**
- 纯叙述或描写的章节不应连续超过3章

### 对话原则

- **每句对话都应该推进情节或揭示人物**
- 避免"信息倾倒"型对话（A向B解释B本应已经知道的事情）
- 使用潜台词——人物说的和想的不一致
- 不同角色有不同的说话风格：

| 人物 | 说话特点 |
|------|---------|
| 林默 | 简洁、直接，较少表达情感 |
| 顾远山 | 措辞正式，善于谈判式表达 |
| 林清寒 | 理性、克制，用问句多于用陈述句 |
| 苏妍 | 专业术语多，兴奋时会加快语速 |
| 宁川 | 思考型，喜欢停顿和反问 |
| 艾琳 | 朴实、有力，善于用生活化的比喻 |
| 沈岚 | 温和但坚定，话不多但每句都有分量 |
| 阿尔瓦 | 平静、精确，偶尔有跨越时间尺度的幽默感 |

## 五、科技描写规范

### 原则

> 科技用于推动思想冲突，而不是单纯展示技术。

### 具体规范

- **只描写角色能感知到的科技**（看到的、听到的、触到的）
- **不解释原理**——除非角色本身需要理解原理来推进剧情
- **尺度感**——星际距离和时间延迟必须真实
- **限制感**——技术有局限，不出现万能解决方案

### 示例

```
✅ 正确：
"通信延迟约两小时。"奥布莱恩看着屏幕说。"我们发送的每一个问题，都要四小时后才能收到回复。"

❌ 错误：
"这个遗迹的能源系统基于零点能提取原理，利用量子涨落来产生无限能量，这是伊洛文明掌握的核心技术之一……"
```

## 六、人物描写规范

### 出场规则

- 核心人物第一次出场时，应有 **2-3段** 的完整描写（外貌、动作、环境关系）
- 之后出场只需1-2句提示性描写
- 非核心角色（如船员）出场时，1段描写即可

### 禁止

- ❌ 直接告诉读者角色的性格（"他是一个勇敢的人"）
- ✅ 通过动作和选择展示性格（探索艇门开启时，他是第一个走出去的）

---

## 七、节奏控制

### 快节奏场景特征

- 短句、短段落
- 对话推进
- 时间感紧迫
- 多行动少心理描写

### 慢节奏场景特征

- 长句、长段落
- 内心独白或观察
- 环境描写丰富
- 时间感松弛

### 节奏切换

- 每卷至少安排2-3个慢节奏章节（角色反思、团队对话、宇宙孤独感的描写）
- 高潮前必须有节奏放缓的积累期
- 不要在连续3章以上的快节奏中保持同一强度——读者会疲劳

---

## 八、风格参考（内参，非模仿对象）

| 维度 | 参考方向 |
|------|---------|
| 宇宙尺度的冷静叙事 | 《2001：太空漫游》式的克制 |
| 多视角群像结构 | 《三体》式的多线推进 |
| 科技描写的精确感 | 《火星救援》式的"用数据说话" |
| 哲学追问的留白 | 《降临》式的"提出问题而非答案" |
| 人物关系的克制 | 《星际穿越》式的"情感通过行动表达" |

## 九、写前自检清单

每章写完后，对照以下问题检查：

- [ ] 本章是否有明确的视角人物？
- [ ] 读者是否只能知道该视角人物知道的信息？
- [ ] 是否有"上帝视角"的段落需要删除？
- [ ] 对话是否推进了情节或揭示了人物？
- [ ] 科技描写是否服务于思想冲突而非展示技术？
- [ ] 本章结尾是否有钩子引导读者进入下一章？
- [ ] 本章的节奏是否符合其在卷中的位置（推进/积累/高潮）？
- [ ] 所有术语使用是否符合 `docs/glossary.md` 规范？
- [ ] **这个事件如何改变了视角人物？**
- [ ] **人物做出的选择如何反映了人类文明面对宇宙未知的某个侧面？**
