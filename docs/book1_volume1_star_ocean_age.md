# 第一卷：《星海时代》

## 定位

章节范围：1-50章

核心主题：

> 人类为什么走向宇宙。

这一卷不是关于发现外星文明，而是展示一个年轻星际文明如何迈出第一步。

人类认为自己正在探索宇宙，但实际上，他们正在被宇宙重新定义。
