# 第一部《赫利俄斯之门》总纲

## 基本定位

主题：
人类文明第一次发现自己并不是宇宙中第一个智慧文明。

核心问题：
> 当一个年轻文明面对比自己古老百万年的文明遗迹时，人类应该选择什么？

第一部目标：
- 建立2300年人类文明背景
- 展示企业推动星际探索
- 引入赫利俄斯事件
- 发现伊洛文明存在
- 保留伊洛文明最终秘密
